git add

1.เข้าที่ไฟล์ ตามชื่อไฟล์ที่เรากำหนด
2.เข้าที่ไฟล์ทั้งหมด
3.เข้าที่ไฟล์ ที่อยู่ในโฟลเดอร์ที่เรากำหนด

git commit

1.บันทึกไฟล์ทั้งหมด ที่พึ่ง git add มาเมื่อกี้
2.บันทึกไฟล์ที่ถูกแก้ไข
3.บันทึกไฟล์ที่ถูกแก้ไข และไฟล์ที่เพิ่มมาใหม่ ไปทั้งหมดพร้อมกัร
4.บันทึกไฟล์ที่ถูกแก้ไข แล้สค่อยบันทึกไฟล์ที่เพิ่มเข้ามาใหม่
5.บันทึกไฟล์ที่ถูกลบ

.gitignore

1.เมินไฟล์ ตามชื่อไฟล์ที่เรากำหนด
2.เมินไฟล์ทั้งหมด ที่อยู่ในโฟลเดอร์ที่เรากำหนด
3.เมินไฟล์จาก Extension ของไฟล์

git reset

1.ลบการกระทำทุกอย่างที่ทำอยู่ ณ ขณะนี้
2.ลบทุกอย่าง และย้อนเวลากลับไปบังความทรงจำที่ต้องการ

git revert

1.ดึงความทรงจำจากอดีตที่ผ่านมา 1 ครั้ง มาแปะไว้ที่ปัจจุบัน
2.ดึงความทรงจำจากอดีตที่ผ่านมา >1 ครั้งมาแปะไว้ที่ปัจจุบัน

git push

1.อัพโหลดโปรเจ็คขึ้นไปเก็บบน Github ครั้งแรก
2.แก้โค้ด และอัพโหลดโปรเจ็คขึ้นไปเก็บบน github อีกที

git pull

1.อัพโหลดโปรแกรมจาก Github มาอัพเดทบนคอม
2.ลองอีกซักรอบ

git clone

1. ดาวโหลดโปรเจ็คจาก Github มาลงใหม่ที่คอมเรา
2.ทดสอบ git push เพื่อให้มั่นใจว่า git clone มาใช้งานได้ไม่มีปัญหาอันใด
